Serverless JSReport Demo

Instructions:

  • Step 1: Get the report's sample data
  • Step 2: Get the report's sample template
  • Step 3: Generate the sample report in various formats

Technical details:

This application interacts with a serverless API with two endpoints:

  1. GET /data returns a knowledge matrix sample
  2. POST /report returns a report which is defined in the payload of the request:
    • data the data object to feed the report
    • template the data object to feed the report
      • content the source code for the report template
      • recipe the JSReport recipe which will determine the report format
      • engine the template engine which degault to handlebars
